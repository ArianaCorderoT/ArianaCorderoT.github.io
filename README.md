# NAME - AI Exoplanet Identifier 🚀\n\n![NAME Banner](https://via.placeholder.com/800x200/8b45ff/ffffff?text=NAME+AI+Exoplanet+Identifier)\n\n## Project Overview\n\nNAME is a cutting-edge web application designed to detect and analyze exoplanets using artificial intelligence. Built by three passionate students from Newton College in Lima, Peru, this platform provides an intuitive interface for exoplanet detection and visualization.\n\n### 🌟 Features\n\n- **Modern Space-Themed UI**: Stunning visual design with animated star backgrounds and glitch effects\n- **File Upload Support**: CSV and FITS file formats with drag-and-drop functionality\n- **Manual Data Entry**: Direct input of time and flux data with validation\n- **Stellar Parameters**: Optional stellar characteristics to enhance analysis\n- **Interactive Visualizations**: Placeholders for transit curves, periodograms, and phase-folded analysis\n- **Planet Library**: Repository of detected exoplanets with search and filtering\n- **Responsive Design**: Optimized for desktop, tablet, and mobile devices\n- **Real-time Notifications**: User-friendly feedback system\n\n## 🎯 Team\n\nWe are three students:\n- One in 10th grade\n- One in 11th grade  \n- One in the first year of university\n\nAll currently studying or having studied at **Newton College** in Lima, Peru.\n\n## 🛠️ Technology Stack\n\n- **Frontend**: HTML5, CSS3, JavaScript (ES6+)\n- **Design**: Custom CSS with Orbitron and Inter fonts\n- **Icons**: Font Awesome 6.0\n- **Animations**: CSS keyframes and transitions\n- **Responsive**: CSS Grid and Flexbox\n\n## 📁 Project Structure\n\n```\nname-exoplanet-detector/\n├── index.html          # Homepage\n├── detector.html       # AI Exoplanet Detector\n├── display.html        # Planet Analysis Results\n├── library.html        # Planet Library\n├── credits.html        # Credits Page\n├── README.md          # Project Documentation\n└── assets/\n    ├── css/\n    │   └── style.css   # Main stylesheet\n    ├── js/\n    │   └── main.js     # JavaScript functionality\n    └── images/         # Image assets\n```\n\n## 🚀 Getting Started\n\n### Prerequisites\n- Modern web browser (Chrome, Firefox, Safari, Edge)\n- Local web server (optional but recommended)\n\n### Installation\n\n1. Clone the repository:\n```bash\ngit clone <repository-url>\ncd name-exoplanet-detector\n```\n\n2. Open with a local server (recommended):\n```bash\n# Using Python\npython -m http.server 8000\n\n# Using Node.js http-server\nnpx http-server\n\n# Using PHP\nphp -S localhost:8000\n```\n\n3. Or simply open `index.html` in your browser.\n\n### Usage\n\n1. **Homepage**: Start at the main page to learn about NAME\n2. **Detector**: Upload CSV/FITS files or enter data manually\n3. **Processing**: Submit data for AI analysis (simulated)\n4. **Results**: View comprehensive analysis on the display page\n5. **Library**: Browse previously analyzed planets\n\n## 🎨 Design Philosophy\n\nNAME features a futuristic space theme with:\n- Deep space color palette (dark blues, purples, teals)\n- Animated starfield background\n- Glitch text effects for the main title\n- Smooth transitions and hover effects\n- Glass-morphism design elements\n- Responsive layout for all devices\n\n## 🔧 Features Implemented\n\n### ✅ Complete\n- [x] Homepage with hero section and navigation\n- [x] File upload with drag & drop\n- [x] Form validation and error handling\n- [x] Manual data entry interface\n- [x] Stellar parameters input\n- [x] Analysis results display page\n- [x] Planet library with search/filter\n- [x] Responsive mobile design\n- [x] Loading states and notifications\n- [x] Smooth page transitions\n\n### 🚧 Future Implementation\n- [ ] Actual AI/ML exoplanet detection algorithms\n- [ ] Real data processing backend\n- [ ] Database integration for planet storage\n- [ ] User authentication system\n- [ ] Advanced data visualization charts\n- [ ] Export functionality for results\n\n## 📱 Browser Support\n\n- ✅ Chrome 80+\n- ✅ Firefox 75+\n- ✅ Safari 13+\n- ✅ Edge 80+\n\n## 🤝 Contributing\n\nThis is a student project from Newton College.
